固定表头
Name Age Address {{data.name}} {{data.age}} {{data.address}}
使用
    固定表头:

    使用 需要指定 column 的 width 属性，否则列头和内容可能不对齐。

  • 1. 在组件ts中定义数据数组
  • // component.ts
    data = [];

    ngOnInit() {
     for (let i = 0; i < 100; i++) {
     this.data.push({
     name : `Edward King ${i}`,
     age : 32,
     address: `London, Park Lane no. ${i}`,
    });
    }
    }
  • 2. 在html中定义标签<nz-table>
  • 3. nz-table组件属性列表:
  • nzDataSource : 表格数据
    nzIsPagination : 是否分页
    nzScroll : 表格的宽高
  • 4. 在nz-table标签中定义标签ng-template，定义表头
  • <ng-template #nzFixedHeader>
    <thead nz-thead>
     <tr>
      <th nz-th [nzWidth]="'150px'"><span>Name</span></th>
      <th nz-th [nzWidth]="'150px'"><span>Age</span></th>
      <th nz-th><span>Address</span></th>
     </tr>
    </thead>
    </ng-template>
表格展开
Name Age Address {{item.name}} {{item.age}} {{item.address}}
使用

    表格展开:

  • 1. 使用nz-tableng-template标签在页面渲染
  • 2. 使用nz-row-indentnz-row-expand-icon标签渲染表格层级关系
  • 3. 在表格层级图表设置点击事件nzExpandChangenzShowExpand
  • <nz-table #nzExpandTable [nzDataSource]="expand_data" [nzIsPagination]="false">
    <thead nz-thead>
    <tr>
    <th nz-th [nzWidth]="40%"><span>Name</span></th>
    <th nz-th [nzWidth]="30%"><span>Age</span></th>
    <th nz-th><span>Address</span></th>
    </tr>
    </thead>
    <tbody nz-tbody>
    <ng-template ngFor let-data [ngForOf]="nzExpandTable.data">
    <ng-template ngFor let-item [ngForOf]="expandDataCache[data.key]">
    <tr nz-tbody-tr *ngIf="(item.parent&&item.parent.expand)||!(item.parent)">
    <td nz-td>
    <nz-row-indent [nzIndentSize]="item.level"></nz-row-indent>
    <nz-row-expand-icon [(nzExpand)]="item.expand" (nzExpandChange)="collapse(expandDataCache[data.key],item,$event)" [nzShowExpand]="!!item.children">
    </nz-row-expand-icon>
    { {item.name} }
    </td>
    <td nz-td>{ {item.age} }</td>
    <td nz-td>{ {item.address} }</td>
    </tr>
    </ng-template>
    </ng-template>
    </tbody>
    </nz-table>
