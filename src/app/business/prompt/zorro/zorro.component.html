NG-ZORRO

    NG ZORRO 致力于提供给 Angular 开发者愉悦的开发体验。

  • 这里是 Ant Design 的 Angular^5.0.0 实现，开发和服务于企业级后台产品。
  • 如果要您使用的是 Angular 4.x 版本，请使用 0.5.x 版本的 NG-ZORRO

    使用

  • 1. 在所需加载模态框组价的Module中，导入NgZorroAntdModule
  • import {NgZorroAntdModule} from 'ng-zorro-antd';
    import {NZ_NOTIFICATION_CONFIG} from 'ng-zorro-antd';
    imports: [
    ...
    NgZorroAntdModule.forRoot(),
    ...
    ],
    providers: [
    { provide: NZ_NOTIFICATION_CONFIG, useValue: { nzTop: '80px' } }
    ]
  • 2. 在component.ts注入NzNotificationService
  • constructor(private _notification: NzNotificationService) {
    }
