使用
    Echarts SVG地图扩展:

    使用

  • 1. 在要使用的模块中导入模块SvgMapModule
  • @NgModule({
    imports: [
    ...
    SvgMapModule
    ],
    declarations: [
    EchartsComponent
    ]
    })
  • 2. 在要创建svgMap的Component.ts中创建所需的属性和回调
  • public svgRoute = 'assets/svg/ningbo.svg';
    public echartMapArea = 'echarts-render-area';
    public divClass = 'text-ee';
    public option;
  • 3. 在Init方法或构造器中通过网路请求或其他方式给otion赋值
  • ngOnInit() {
    this.option = {
    backgroundColor: '#eee',
    height: '100px',
    ...
    }
    }
  • 3. 在需要加载MAP的html模板，添加:
  • <app-svg-map
    [option]="option"
    [divClass]="divClass"
    [svg]="svgRoute"
    [selector]="echartMapArea"
    (clickCallBack)="mapClickCallBack($event)"></app-svg-map>

  • 3. 创建assets/js/scripts.d.ts文件，声明init对象
  • 4. 创建组件 svg-map 组件

  • ng g c svg-map

  • 4. 创建组件 svg-map module,在module中declarations 并导出 SvgMapComponent
  • @NgModule({
    imports: [
    CommonModule,
    FormsModule
    ],
    declarations: [
    SvgMapComponent
    ],
    exports: [
    SvgMapComponent
    ]
    })
