复合图表-柱状 & 折现 &区域

he most comprehensive JavaScript charting library, with over 90+ charts and 1000+ maps. Loved by 27,000 customers in 118 countries.

The Angular4-FusionCharts component is a simple and lightweight binding for the FusionCharts charting library that lets you add rich and interactive charts to your Angular projects.

样例
使用
    Prominent features of this component include:
  • 1. Adding a chart using only a single Angular component
  • 2. Auto-updating the chart object when component properties are modified
  • 3. Adding a chart by fetching data from a JSON or XML URL or by using the component props binding
  • 4. Enabling interactivity between JavaScript charts
  • 5. Offering advanced control by giving access to the complete FusionCharts object that contains the chart configuration

    使用

  • 1. 安装FunsionChart
  • sudo cnpm install fusioncharts --save
  • 2. 安装angular4-fusioncharts
  • sudo cnpm install fusioncharts --save
  • ## 或者使用下面的命令执行上面两条
  • npm install fusioncharts angular4-fusioncharts --save
  • 3. 在需要加载funsioncharts的模块import angular4-funsioncharts
  • // Module
    import * as FusionCharts from 'fusioncharts';
    import * as Charts from 'fusioncharts/fusioncharts.charts';
    import * as FintTheme from 'fusioncharts/themes/fusioncharts.theme.fint';

    mport { FusionChartsModule } from 'angular4-fusioncharts';

    @NgModule({
    ...
    imports: [
    ...
    FusionChartsModule.forRoot(FusionCharts, Charts, FintTheme)
    ],
    ...
    })
  • 4. 在Component中添加图表的属性配置
  • export class AppComponent {
    width = 600;
    height = 400;
    type = 'mscombi2d';
    dataFormat = 'json';
    dataSource;
    constructor() {
      this.dataSource = {
        ... // 数据源的配置可以在Api中查询
    }
    }
  • 5. 在html中使用标签<fusioncharts>渲染图表
  • <fusioncharts
    [width]="width"
    [height]="height"
    [type]="type"
    [dataFormat]="dataFormat"
    [dataSource]="dataSource"
    ></fusioncharts>
