使用

    BootStrap 分页组件

  • 1. 在需要分页组件的模块module.ts文件中，导入PaginationModule
  • import { PaginationModule } from 'ngx-bootstrap/pagination';
    ...
    imports: [PaginationModule.forRoot(),...]
  • 2. 在template中定义pagination渲染分页组件, 并在Component定义回调方法pageChanged
  • <pagination
      [totalItems]="63"
      [(ngModel)]="currentPage"
      (pageChanged)="pageChanged($event)"></pagination>
  • 3. 其他配置
  • 详细配置 https://valor-software.com/ngx-bootstrap/#/pagination
